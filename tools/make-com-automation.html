Getting Started with Make.com

Free Tier Limit: 100 operations/month. Consider the Core plan ($9/month) for 1,000 operations.

Step 1: Create Make.com Account

Sign up at make.com

  • Start with free tier (100 operations/month)
  • Connect your email (Gmail/Outlook)
  • Connect your phone (Twilio for SMS)
  • Connect Cal.com for calendar integration

Step 2: Set Up Webhook Endpoint

Create a webhook to receive data from your website:

Webhook URL: https://hook.make.com/YOUR_WEBHOOK_KEY
Method: POST
Headers: Content-Type: application/json

Step 3: Install Website Webhook Script

Add this to your contact forms to trigger automations:

// Add to your contact form submission handler async function submitToMake(formData) { const response = await fetch('https://hook.make.com/YOUR_WEBHOOK_KEY', { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ timestamp: new Date().toISOString(), source: 'watts_website', lead: { name: formData.name, phone: formData.phone, email: formData.email, service: formData.service, message: formData.message, urgency: formData.timeline }, metadata: { page: window.location.pathname, userAgent: navigator.userAgent, referrer: document.referrer } }) }); return response.ok; }

Automation Workflows

New Lead Automation

Trigger: Webhook receives new lead from website
Action 1: Send SMS to Justin: "New lead: [Name] - [Service] - Call [Phone]"
Action 2: Add lead to Google Sheets (CRM tracker)
Action 3: Send auto-email to client: "Thanks for reaching out! We'll call within 2 hours."
Action 4: Create task in Asana/Trello for follow-up
Action 5: If service="Accessibility", create PandaDoc contract draft

Booking Confirmation Automation

Trigger: Cal.com new appointment booked
Action 1: Send confirmation SMS with appointment details
Action 2: Add to Google Calendar with reminder 1 day before
Action 3: Check weather forecast 24h before appointment
Action 4: If bad weather, send rescheduling options

Bid Follow-up Automation

Trigger: New bid created (daily check at 9 AM)
Action 1: Send bid email with PDF attachment
Action 2: Schedule follow-up in 3 days if no response
Action 3: After 7 days, send final follow-up with discount offer
Action 4: Mark as "cold lead" after 14 days

Monitoring & Analytics

Pro Tip: Create a dashboard to track automation performance

Key Metrics to Track:

  • Lead Response Time: Average time from submission to first contact
  • Conversion Rate: Leads ‚Üí Appointments ‚Üí Signed Contracts
  • Automation Success Rate: Failed scenarios and error logs
  • Operation Usage: Stay within Make.com limits
  • ROI per Channel: Track which marketing brings best leads

Security & Best Practices

  • Secure Webhooks: Use unique URLs and verify source
  • Data Privacy: Don't store sensitive data in Make logs
  • Error Handling: Set up notifications for failed scenarios
  • Rate Limiting: Stagger API calls to avoid limits
  • Backup Plans: Manual processes for critical functions
  • Documentation: Keep a map of all active automations
