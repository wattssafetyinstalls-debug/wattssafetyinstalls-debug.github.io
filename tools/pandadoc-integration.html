Quick Setup Guide

Important: PandaDoc API requires a paid plan. The free tier allows manual document creation but not API access.

Step 1: Create PandaDoc Account

Sign up at pandadoc.com

  • Choose Business plan ($19/month) for API access
  • Verify your email and complete profile
  • Upload your business logo

Step 2: Get API Credentials

Navigate to Settings → Integrations → API

  • Copy your API Key (starts with "pdoc_")
  • Note your Sandbox vs Production keys
  • Set webhook URL: https://wattsatpcontractor.com/tools/webhooks/pandadoc

Step 3: Create Document Templates

Create reusable templates for common documents:

  • Service Agreement / Contract
  • Change Order Form
  • Bid Proposal
  • Invoice Template

Step 4: Store API Key Securely

Never store API keys in client-side code. Use environment variables:

# .env file (NEVER commit to git) PANDADOC_API_KEY=pdoc_your_api_key_here PANDADOC_ENVIRONMENT=production

Implementation Examples

Send Contract for Signature

// Node.js example const pandadoc = require('pandadoc-api-sdk'); async function sendContract(clientEmail, clientName, bidAmount) { const templateId = 'your_template_id_here'; const document = await pandadoc.documents.create({ name: `Service Agreement - ${clientName}`, template_uuid: templateId, recipients: [ { email: clientEmail, first_name: clientName.split(' ')[0], last_name: clientName.split(' ')[1] || '', role: 'client' } ], tokens: { client_name: clientName, bid_amount: `$${bidAmount}`, contractor_name: 'Watts ATP Contractor', license_number: '#54690-25', date: new Date().toLocaleDateString() }, metadata: { source: 'watts_website', bid_amount: bidAmount } }); return document; }

Webhook Handler (Express.js)

// Handle PandaDoc webhooks app.post('/webhooks/pandadoc', (req, res) => { const { status, data } = req.body; if (status === 'document.completed') { // Document was signed! const { id, metadata } = data.document; // Update your database await updateBidStatus(metadata.bid_id, 'contract_signed'); // Send confirmation email await sendConfirmationEmail(data.recipients[0].email); // Notify Justin await sendSMS('(405) 410-6402', `Contract signed: ${data.name}`); } res.status(200).send('OK'); });

Security & Best Practices

  • API Key Security: Never expose in frontend code
  • Webhook Verification: Always verify webhook signatures
  • Document Retention: Keep signed documents for 7 years (NE law)
  • Backup Strategy: Download signed PDFs to local storage
  • Access Control: Limit template editing to authorized users
// Verify webhook signature const crypto = require('crypto'); function verifyWebhook(req) { const signature = req.headers['x-pandadoc-signature']; const secret = process.env.PANDADOC_WEBHOOK_SECRET; const payload = JSON.stringify(req.body); const expectedSignature = crypto .createHmac('sha256', secret) .update(payload) .digest('hex'); return signature === expectedSignature; }
